<ul>
  <li><b>Frontend:</b> HTML5, CSS3, JavaScript (React.js)</li>

<hr />

<p>Follow these steps to run the development server locally:</p>

<ol>
  <li><strong>Clone the repository:</strong>
    <pre><code>git clone https://github.com/kaniapaulina/my-webfolio.git</code></pre>
  </li>
  <li><strong>Navigate into the directory:</strong>
    <pre><code>cd my-webfolio</code></pre>
  </li>
  <li><strong>Install dependencies:</strong>
    <pre><code>npm install</code></pre>
  </li>
  <li><strong>Start the development server:</strong>
    <pre><code>npm start</code></pre>
    <em>(Note: If you used Vite, use <code>npm run dev</code> instead)</em>
  </li>
</ol>
